# Pandas Code

This repository contains a **Python (Pandas) starter project** for data analysis. It demonstrates basic data cleaning, exploration, and analysis using Pandas, along with simple visualization using Matplotlib.

## Features
- Load and inspect datasets (`.csv`)
- Handle missing values and duplicates
- Perform basic analysis (total revenue, top products)
- Save cleaned datasets
- Simple data visualization (line plots)

## Getting Started

### Prerequisites
- Python 3.x
- Pandas library
- Matplotlib library (for plotting)

### Installation
Install required libraries via terminal:
```bash
pip install pandas matplotlib
